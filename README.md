## 🎯 Overview
AI Request Desk is an internal tool that helps organizations streamline access, track usage, and manage approvals for AI tools in one unified platform.
**Live Demo:** [ai-consulting--ankur0609.replit.app](https://ai-consulting--ankur0609.replit.app)
---
## ✨ Features
- **Request Management** - Submit and track AI tool requests
- **Multi-Department Support** - Sales, Marketing, Customer Success, Product teams
- **Usage Tracking** - Monitor AI tool usage across the organization
- **Approval Workflow** - Manage approvals for AI resources
- **User Authentication** - Secure sign-in system
- **AI Agent Integration** - Support for multiple AI agents
